Visualize Bootstrap v3.3.x Media Query Viewport Breakpoints

1.) Extra small screen. Phone. xs. No media query. Styles applied to content at all viewport widths.

2.) Small screen. Tablet. sm. @media (min-width: 768px) { ... }. Styles applied to content when viewport width is 768+ pixels.
Width 767px.

3.) Medium screen. Desktop. md. @media (min-width: 992px) { ... }. Styles applied to content when viewport width is 992+ pixels.
Width 991px.

4.) Large screen. Wide desktop. lg. @media (min-width: 1200px) { ... }. Styles applied to content when viewport width is 1200+ pixels.
Width 1199px.
